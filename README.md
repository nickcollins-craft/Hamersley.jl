[![Aqua](https://raw.githubusercontent.com/JuliaTesting/Aqua.jl/master/badge.svg)](https://github.com/JuliaTesting/Aqua.jl)

Hamersley is a package designed to facilitate finite element analyses for problems in higher order and multiphysics solid mechanics, with a particular eye towards geomechanics. It is a spiritual successor to [Numerical Geolab](https://github.com/AlexSTA1993/numerical_geolab), and aims to achieve much of the same functionality, while adding additional flexibility and quality-of-life improvements.

## Installation

Hamersley is not currently in the Julia General Registry, so it can only be installed by cloning this repository.
